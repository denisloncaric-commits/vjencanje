✦ Google tablica za RSVP

Sve potvrde dolaska spremaju se u tvoju Google tablicu.

  1. Kreiraj tablicu na sheets.new
  2. Prvi red (headeri): id, firstName, lastName, email, status, adults, children, message, timestamp
  3. Preimenuj list u RSVP
  4. Otvori script.google.com, novi projekt
  5. Zamijeni kod s ovim:
function doGet(e) { const s = SpreadsheetApp.openById('ID').getSheetByName('RSVP'); const d = s.getDataRange().getValues(); if (d.length<2) return ContentService .createTextOutput(JSON.stringify({data:[]})) .setMimeType(ContentService.MimeType.JSON); const h = d[0]; const data = d.slice(1).map(r => { let o = {}; h.forEach((hi, i) => o[hi] = r[i]); return o; }); return ContentService .createTextOutput(JSON.stringify({data})) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const s = SpreadsheetApp.openById('ID').getSheetByName('RSVP'); const b = JSON.parse(e.postData.contents); s.appendRow([ b.id, b.firstName, b.lastName, b.email, b.status, b.adults, b.children, b.message || '', b.timestamp ]); return ContentService .createTextOutput(JSON.stringify({ok: true})) .setMimeType(ContentService.MimeType.JSON); }
  1. Kopiraj ID tablice iz URL-a (npr. 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms)
  2. Zamijeni 'ID' u kodu sa svojim ID-jem
  3. Spremi (Ctrl+S) → Deploy → New deployment → Web app
  4. Execute as: Me  |  Access: Anyone
